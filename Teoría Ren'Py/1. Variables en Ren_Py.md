## Variables en Ren'Py
---
En Ren'Py, las variables son fundamentales porque son las que permiten que tu novela visual deje de ser una historia lineal y se convierta en una experiencia donde las decisiones del jugador realmente importan.

Aquí tienes una guía rápida para entender cómo funcionan, desde lo más básico hasta las mejores prácticas.

**1.  Declaración de Variables: default vs define**

Esta es la distinción más importante en Ren'Py. Si te equivocas aquí, podrías tener problemas al guardar la partida.

```
default: Se usa para variables que van a cambiar durante el juego (puntos de afecto, el nombre del jugador, si un objeto fue recogido).

define: Se usa para valores que son constantes o que no cambian (personajes, configuraciones fijas).
```

Fragmento de código
```python
# Esto es una constante (no cambia)

define e = Character("Eileen")

# Esto es una variable (va a cambiar)

default puntos_amor = 0  
default tiene_llave = False  
default nombre_jugador = "Prota"
```

**2.  Tipos de Datos Comunes**

En Ren'Py (que usa Python de base), las variables pueden ser de varios tipos:

| Tipo | Ejemplo | Uso común |
| --- | --- | --- |
| Booleano | True / False | Interruptores (¿Hablaste con ella? Sí/No) |
| Entero (Int) | 10, -5 | Contadores de puntos, dinero, edad. |
| Flotante (Float) | 0.5, 3.14 | Porcentajes o barras de energía. |
| Cadena (String) | ¡Hola! | Nombres de personajes o textos dinámicos. |

  
**3. Modificar Variables en el Script**

Para cambiar el valor de una variable durante la historia, usamos la línea de Python de una sola línea mediante el símbolo \$.  
Fragmento de código
```python
label start:  
"Te encuentras una moneda en el suelo."

# Sumamos 1 a la variable
$ monedas += 1

"Ahora tienes [monedas] monedas."

if monedas > 5:
    "Eres rico."
else:
    "Aún eres pobre."
```

**4.  Mostrar Variables en el Texto**

Para que el jugador vea el valor de una variable dentro de un diálogo, usamos los corchetes \[\]. Ren'Py sustituirá el nombre de la variable por su valor actual.

```python
"Hola, [nombre_jugador]. Tienes [puntos_amor] puntos conmigo."
```

**5.  El Menú de Decisiones**

Las variables suelen brillar cuando se combinan con el comando menu. Aquí es donde guardas la elección del usuario.  
Fragmento de código
```python
menu:  
"¿Qué quieres hacer?"


"Ayudar a Sarah":
    $ puntos_sarah += 1
    "Sarah te sonríe."
    
"Ignorar a Sarah":
    $ puntos_sarah -= 1
    "Sarah se ve decepcionada."
```

## Nota importante:

Usa siempre nombres de variables claros. Es mejor llamar a una variable puntos_amistad_elena que simplemente p. Tu "yo" del futuro te lo agradecerá cuando el código tenga miles de líneas.
