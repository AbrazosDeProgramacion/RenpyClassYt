# entre pantallas, saves y loads.

    "¡Hola! Tienes [monedas] monedas."

    # Simulamos que el jugador encuentra una moneda
    $ monedas += 1

    "¡Has encontrado una moneda! Ahora tienes [monedas] monedas."

    # Si aquí guardas y cargas el juego, 
    # 'default' se asegura de que el valor '1' se mantenga.
    # Si hubieras usado 'define', esto causaría un error 
    # o se resetearía a su valor inicial.
    return
